# Ubeeps 優生醫療 (Static Website)

這是一個關於幹細胞治療與再生醫學的靜態網站專案。

## 專案簡介
Ubeeps 致力於推動再生醫學資訊透明化，連結病患與合規醫療資源。本網站提供幹細胞科學知識、臨床應用案例及最新消息。

## 技術架構
- **Frontend**: 純 HTML5, Tailwind CSS (透過 CDN 引入)
- **Interactive**: Vanilla JavaScript (無須編譯)
- **Features**:
  - 響應式設計 (RWD)
  - 浮動式 Chatbot (整合 WhatsApp 與 Email)
  - 靜態頁面導航

## 部署說明 (GitHub Pages)
本專案已最佳化，可直接部署於 GitHub Pages：
1. 將程式碼推送到 GitHub Repository。
2. 進入 Settings > Pages。
3. Source 選擇 `main` branch 並儲存。
4. 網站即可透過 `https://<your-username>.github.io/<repo-name>/` 訪問。

## 目錄結構
- `/`: 主要 HTML 頁面 (index.html, about.html, etc.)
- `/js`: JavaScript 邏輯 (chatbot.js)
- `/images`: 本地圖片資源
